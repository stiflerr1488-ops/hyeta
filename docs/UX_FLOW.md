# UX / Сценарий

## Экраны
1. Верхняя панель: открыть проект, сохранить, отмена/повтор, поиск.
2. Боковая панель: дерево файлов + быстрый переход к HTML.
3. Центр: живое превью в iframe.
4. Правая панель: свойства выбранного элемента.
5. Панель поиска: результаты совпадений по проекту.

## Основной сценарий
1. Открыть проект → выбор директории.
2. Автоиндексация → отображение дерева и страниц.
3. Выбор HTML-страницы → загрузка в превью.
4. Клик в превью → данные элемента в панели свойств.
5. Изменение текста/атрибутов → применить.
6. Ctrl+S сохраняет файл.
7. Ctrl+Z / Ctrl+Y откатывает/повторяет.
8. Поиск по проекту (Ctrl+Shift+F).

## Экспорт/публикация
- MVP: экспорт ZIP (этап дорожной карты, интерфейс уже заложен архитектурно).
- v1: публикация в SFTP/FTP.
