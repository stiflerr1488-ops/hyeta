# UX / Flow

## Экраны
1. Toolbar: открыть проект, save, undo/redo, поиск.
2. Sidebar: дерево файлов + быстрый переход к HTML.
3. Center: live preview в iframe.
4. Right panel: свойства выбранного элемента.
5. Search panel: результаты совпадений по проекту.

## Основной сценарий
1. Open Project → выбор директории.
2. Авто-индексация → отображение tree + pages.
3. Выбор HTML-страницы → загрузка в preview.
4. Клик в preview → данные элемента в панели свойств.
5. Изменение текста/атрибутов → Apply.
6. Ctrl+S сохраняет файл.
7. Ctrl+Z / Ctrl+Y откатывает/повторяет.
8. Поиск по проекту (Ctrl+Shift+F).

## Экспорт/деплой
- MVP: экспорт ZIP (roadmap step, интерфейс подготовлен архитектурно).
- v1: публикация в SFTP/FTP.
