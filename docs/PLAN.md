# План разработки

## Спринт 1 (Основа MVP)
- Задачи: импорт директории, индексатор файлов, дерево, список HTML, превью.
- Результат: проект открывается и отображается локально.
- Критерий готовности: страницы открываются, относительные пути работают.
- Риски: безопасность превью, большие директории.

## Спринт 2 (Редактируемый MVP)
- Задачи: клик по элементу, встроенное редактирование текста, атрибуты href/src/alt/title, сохранение, отмена/повтор.
- Результат: базовые контентные правки без кода.
- Критерий готовности: правки применяются и сохраняются в файлы.
- Риски: точное соответствие выбранного DOM-элемента исходнику.

## Спринт 3 (Качество MVP+)
- Задачи: поиск/замена по проекту, простое предпросмотр-сравнение, экспорт ZIP, смоук-тесты.
- Результат: завершённый сценарий до передачи на хостинг.
- Критерий готовности: редактирование + проверка + экспорт в одном приложении.
- Риски: производительность на 1000+ файлов.
