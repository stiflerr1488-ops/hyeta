# План разработки

## Sprint 1 (MVP Foundation)
- Задачи: импорт директории, индексатор файлов, дерево, список html, preview.
- Outcome: проект открывается и рендерится локально.
- DoD: страницы открываются, относительные пути работают.
- Риски: безопасность preview, большие директории.

## Sprint 2 (Editable MVP)
- Задачи: клик по элементу, inline текст, атрибуты href/src/alt/title, save, undo/redo.
- Outcome: базовые контентные правки без кода.
- DoD: правки применяются и сохраняются в файлы.
- Риски: точное соответствие выбранного DOM-элемента исходнику.

## Sprint 3 (MVP+ Quality)
- Задачи: поиск/замена по проекту, простые diff-preview, экспорт ZIP, smoke tests.
- Outcome: завершенный flow до передачи на хостинг.
- DoD: редактирование + проверка + экспорт в одном приложении.
- Риски: производительность на 1000+ файлов.

## v1
- SFTP/FTP publish.
- Плагинные «фишки» с reversible patch preview.
- История между сессиями.
- Опциональная Git интеграция.
